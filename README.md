# Java Abstraction - Vaccine Management System

## Overview

The Vaccine Management System is a Core Java project developed to demonstrate the concept of **Abstraction** in Object-Oriented Programming (OOP). The project uses an abstract class with concrete methods and extends it through a subclass to simulate a simple vaccination process based on user eligibility.

---

## Features

- Checks user eligibility for vaccination.
- Uses an abstract class to define the vaccination process.
- Implements concrete methods for the first and second vaccine doses.
- Demonstrates inheritance and method implementation.
- Calculates the vaccination fee after successful eligibility.

## How to Run

1. Clone or download this repository.
2. Open the project in Eclipse or any Java IDE.
3. Compile all Java files.
4. Run the `VaccinationSuccessful.java` file.
5. Enter the required details to check vaccination eligibility.
